# Simple Calculator

A basic calculator built with a simple UI that performs addition, subtraction, multiplication, division, and displays the result. Perfect for beginners to understand JavaScript logic and DOM manipulation.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Equal (=) to get the result
- Simple and intuitive user interface

## Demo

![Calculator][calculator.png]

## How to Use

1. Open the index.html file in your browser.
2. Click the buttons to perform calculations.
3. Press = to see the result.

## Code Highlights

- Handles basic arithmetic operations.
- Updates display dynamically on each button click.
- Easy to extend for more advanced features in future.
