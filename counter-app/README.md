# Counter App

A simple counter application that allows users to increase, decrease, and reset a numeric value.

## Features
- Increment counter
- Decrement counter
- Reset counter to zero
- Instant UI update on button click
- Persistent storage using `localStorage`

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
1. Clone the repository
2. Open `index.html` in any modern browser

## How It Works
- The counter value is stored in a JavaScript variable
- Button click events update the value
- DOM is updated immediately after each action
